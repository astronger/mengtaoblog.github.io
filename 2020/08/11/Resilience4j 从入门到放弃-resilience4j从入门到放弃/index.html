简介

Resilience4j 是 Spring Cloud Greenwich 版推荐的容错解决方案，相比 Hystrix ，Resilience4j 专为 Java8 以及函数编程而设计。在 Resilience4j 中你用什么可直接添加什么以来就行。
Resilience4j 主要有以下功能：

  1. CircuitBreaker(熔断器)
  2. RateLimiter(限流)
  3. Retry(请求重试)
  4. 限时
  5. 缓存
  6. 信号量的隔离
