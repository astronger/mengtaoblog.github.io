最近在看江南一点雨的教程以及视频，以下总结的笔记资料等将会参考他的为主，声明：仅供自己学习参考使用。

总结一下Spring Boot的主要优点：

为所有Spring开发者更快的入门

开箱即用，提供各种默认配置来简化项目配置

内嵌式容器简化Web项目

没有冗余代码生成和XML配置的要求
